Setup

The person-level file (Persona_CPV-2024.csv) is ~3 GB with 11,365,333 rows × 118 columns, so we use the arrow package to query it lazily via dplyr without loading it into memory. Only call collect() after filtering/aggregating to a manageable size.

The codebook is parsed from BOL-INE-CPV-2024.xml (DDI format, 184 variables).

Indigenous self-identification (p32)

There are three variables covering indigenous/afro-bolivian identity:

Variable Type Description
p32_pueblo_per Raw Yes/No: self-identifies with an indigenous/afro-bolivian group (1 = Sí, 2 = No, 9 = Sin respuesta)
p32_pueblo_cod Raw Specific group code as written/transcribed; 145 categories including generic terms (9xx) and foreign groups (5xx)
p32_pueblos Derived CTAI-consolidated reference list; 58 categories, re-coding subgroups into broader groups and generic terms into “Otras declaraciones”

The 9xx codes in p32_pueblo_cod represent respondents who said “Yes” but used a generic term rather than a specific group name: 905 = Campesino, 903 = Indígena, 904 = Originario, etc. These are reclassified to “Otras declaraciones” in the derived p32_pueblos.

Alignment: does language match identity?

For each indigenous-language speaker, p32_pueblos responses are classified as:

  • Matching group — identity name corresponds to the language (e.g. Quechua speaker → Quechua identity)
  • Different group — identifies with a named but different group
  • Generic identity — said “Sí” but used a generic term (→ “Otras declaraciones” in p32_pueblos)
  • Does not identify — p32_pueblos == "No se autoidentifica"

Two apparent mismatches are actually coherent: Kabineña speakers identify as Cavineño (same group, different spelling conventions), and Zamuco speakers identify as Ayoreo (Zamuco is the language name; Ayoreo is the ethnonym used in identity contexts).

classify_alignment2 <- function(idioma, pueblos) {
  equiv <- c("Kabineña" = "Cavineño", "Zamuco" = "Ayoreo")
  expected <- coalesce(equiv[idioma], idioma)

    str_detect(pueblos, fixed(expected, ignore_case = TRUE)) |
      str_detect(expected, fixed(pueblos, ignore_case = TRUE))  ~ "Matching group",
    pueblos == "No se autoidentifica"                           ~ "Does not identify",

    pueblos == "Otras declaraciones"                            ~ "Generic identity",
    !is.na(pueblos)                                             ~ "Different group",
    TRUE                                                        ~ "No identity data"
  )
}

make_heatmap <- function(data, strat_col, strat_val, title, show_y = TRUE) {
  d <- data |> filter(.data[[strat_col]] == strat_val)
  p <- ggplot(d, aes(x = age_group, y = mat_label, fill = pct)) +
    geom_tile(color = "white", linewidth = 0.4) +
    geom_text(aes(label = paste0(pct, "%")), size = 2.5) +
    scale_fill_distiller(palette = "RdYlBu", direction = 1,
                         limits = c(0, 100), name = "Retention\n(%)") +
    labs(title = title, x = NULL, y = NULL) +
    theme_minimal(base_size = 10) +
    theme(panel.grid  = element_blank(),
          plot.title  = element_text(size = 10, face = "bold"),
          legend.key.height = unit(0.8, "cm"))
  if (!show_y) p <- p + theme(axis.text.y = element_blank())
  p
}

Rural vs Urban

(make_heatmap(hm_urban, "urbrur_label", "Rural", "Rural", show_y = TRUE) |
 make_heatmap(hm_urban, "urbrur_label", "Urban", "Urban", show_y = FALSE)) +
  plot_annotation(
    title    = "Language retention by age: Rural vs Urban",
    subtitle = "% of indigenous mother-tongue speakers using same language daily | full census",
    theme    = theme(plot.title    = element_text(face = "bold"),
                     plot.subtitle = element_text(size = 9))
  ) +
  plot_layout(guides = "collect") &
  theme(legend.position = "right")

Urban retention is substantially lower for nearly every language and every age group. The most dramatic contrasts are in Quechua (rural 0–14: 90%, urban 0–14: 52%) and Aymara (rural 0–14: 87%, urban 0–14: 41%), showing that urban language shift begins early in childhood. Tsimane´ is the notable exception: even urban Tsimane´ children retain at 77%, suggesting that community ties or residential clustering maintain the language even in city settings. In rural areas, most languages show the expected U-shape (high retention among children and elders, dip among working-age adults); the urban panels flatten this pattern, with low retention sustained across all ages.

Tacana and Mojeño Ignaciano stand out: both show near-zero or very low retention among 45–59 year-olds in rural areas (60% and 29% respectively), indicating that language shift among these groups pre-dates urbanisation and reflects longer-term community-level decline.

Self-identifies vs Does not identify

(make_heatmap(hm_identity, "identifies", "Self-identifies",  "Self-identifies",   show_y = TRUE) |
 make_heatmap(hm_identity, "identifies", "Does not identify","Does not identify", show_y = FALSE)) +
  plot_annotation(
    title    = "Language retention by age: Self-identifies as indigenous vs Does not identify",
    subtitle = "% of indigenous mother-tongue speakers using same language daily | full census",
    theme    = theme(plot.title    = element_text(face = "bold"),
                     plot.subtitle = element_text(size = 9))
  ) +
  plot_layout(guides = "collect") &
  theme(legend.position = "right")

Self-identifiers consistently retain at higher rates. The “Does not identify” panel shows substantially lower retention across all languages, but the pattern is noisier — several cells for smaller languages (Kabineña, Tacana, Mojeño Ignaciano) are based on very small counts and should be interpreted with caution.

For the largest groups, the gap is substantial and age-graded. Among Quechua speakers aged 15–29: self-identifiers retain at 66%, non-identifiers at 47%. For Aymara: 65% vs 45%. This suggests that ethnic identity and daily language use are mutually reinforcing — people who have shifted to Spanish in daily life are also less likely to self-identify, and vice versa — though causality cannot be established from cross-sectional data.

Among the most isolated groups (Tsimane´, Weenhayek, Zamuco, Uru-Chipaya), the “Does not identify” column remains relatively high (60–100%), consistent with community-level language maintenance that is independent of individual identity choices. Kabineña non-identifiers aged 30–44 and 45–59 show 0% retention (very small n), while self-identifiers in those cohorts retain at 47–80% — the sharpest identity-linked divergence in the dataset.
