Indigenous self-identification (p32)

There are three variables covering indigenous/afro-bolivian identity:

Variable Type Description
p32_pueblo_per Raw Yes/No: self-identifies with an indigenous/afro-bolivian group (1 = Sí, 2 = No, 9 = Sin respuesta)
p32_pueblo_cod Raw Specific group code as written/transcribed; 145 categories including generic terms (9xx) and foreign groups (5xx)
p32_pueblos Derived CTAI-consolidated reference list; 58 categories, re-coding subgroups into broader groups and generic terms into “Otras declaraciones”

The 9xx codes in p32_pueblo_cod represent respondents who said “Yes” but used a generic term rather than a specific group name: 905 = Campesino, 903 = Indígena, 904 = Originario, etc. These are reclassified to “Otras declaraciones” in the derived p32_pueblos.

A lookup table for p32_pueblo_cod is saved at pueblo_cats.rds (145 rows). The p32_pueblos codes follow a re-numbered scheme (1–58) documented in the variable dictionary.

Alignment: does language match identity?

For each indigenous-language speaker, p32_pueblos responses are classified as:

  • Matching group — identity name corresponds to the language (e.g. Quechua speaker → Quechua identity)
  • Different group — identifies with a named but different group
  • Generic identity — said “Sí” but used a generic term (→ “Otras declaraciones” in p32_pueblos)
  • Does not identify — p32_pueblos == "No se autoidentifica"

Two apparent mismatches are actually coherent: Kabineña speakers identify as Cavineño (same group, different spelling conventions), and Zamuco speakers identify as Ayoreo (Zamuco is the language name; Ayoreo is the ethnonym used in identity contexts).

classify_alignment2 <- function(idioma, pueblos) {
  equiv <- c("Kabineña" = "Cavineño", "Zamuco" = "Ayoreo", "Bésiro" = "Chiquitano")
  expected <- coalesce(equiv[idioma], idioma)
  case_when(
    str_detect(pueblos, fixed(expected, ignore_case = TRUE)) |
      str_detect(expected, fixed(pueblos, ignore_case = TRUE))  ~ "Matching group",
    pueblos == "No se autoidentifica"                           ~ "Does not identify",
    pueblos == "Sin respuesta"                                  ~ "No response",
    pueblos == "Otras declaraciones"                            ~ "Generic identity",
    !is.na(pueblos)                                             ~ "Different group",
    TRUE                                                        ~ "No identity data"
  )
}

label_compact <- function(x) {
  case_when(
    abs(x) >= 1e6 ~ paste0(round(abs(x) / 1e6, 1), "M"),
    abs(x) >= 1e3 ~ paste0(round(abs(x) / 1e3, 1), "K"),
    TRUE           ~ as.character(abs(x))
  )
}

make_count_diverging <- function(df, lvls_right, lvls_left, cols, title,
                                 subtitle, x_label, arrow_left, arrow_right,
                                 base_size = 11) {
  # Font sizes scale with base_size
  seg_label_size  <- base_size * 0.23
  total_label_size <- base_size * 0.27

  # Factor order: right categories (inner → outer), then left (inner → outer)
  df <- df |>
    mutate(outcome_plot = factor(outcome_plot, levels = c(lvls_right, lvls_left)))

  # Compute label positions (desc: ggplot2 stacks last level closest to axis)
  seg_labs <- df |>
    arrange(idioma_grp, side, desc(outcome_plot)) |>
    group_by(idioma_grp, side) |>
    mutate(
      abs_cum   = cumsum(n),
      abs_start = abs_cum - n,
      mid       = if_else(side == "left",
                          -(abs_start + abs_cum) / 2,
                           (abs_start + abs_cum) / 2)
    ) |>
    ungroup() |>
    # Label segments > 20k, > 3% of total, and < 85% (dominant segments are obvious)
    filter(n > 20000, n / total > 0.03, n / total < 0.85) |>
    mutate(label = label_compact(n))

  # Right-side total (identifies) for end-of-bar labels
  right_totals <- df |>
    filter(side == "right") |>
    group_by(idioma_grp) |>
    summarise(right_total = sum(n), .groups = "drop")

  # Y-axis labels: language name + total on a second line
  y_labels <- df |>
    group_by(idioma_grp) |>
    summarise(total = first(total), .groups = "drop") |>
    mutate(y_lab = paste0(idioma_grp, "\nn = ", scales::comma(total))) |>
    pull(y_lab, name = idioma_grp)

  df <- df |>
    mutate(idioma_grp = factor(y_labels[as.character(idioma_grp)],
                               levels = y_labels))

  right_totals <- right_totals |>
    mutate(idioma_grp = factor(y_labels[as.character(idioma_grp)],
                               levels = y_labels))
  seg_labs <- seg_labs |>
    mutate(idioma_grp = factor(y_labels[as.character(idioma_grp)],
                               levels = y_labels))

  ggplot(df, aes(x = n_plot, y = idioma_grp, fill = outcome_plot)) +
    geom_col(width = 0.7) +
    geom_text_repel(
      data = seg_labs,
      aes(x = mid, y = idioma_grp, label = label),
      inherit.aes = FALSE, size = seg_label_size, fontface = "bold",
      direction = "x", nudge_y = 0.48,
      segment.size = 0.25, segment.color = "grey50",
      min.segment.length = 0,
      box.padding = 0.2, force = 3, max.overlaps = 20
    ) +
    geom_text(
      data = right_totals,
      aes(x = right_total * 1.02, y = idioma_grp,
          label = scales::comma(right_total)),
      inherit.aes = FALSE, hjust = 0, size = total_label_size, fontface = "bold"
    ) +
    geom_vline(xintercept = 0, linewidth = 0.4) +
    scale_x_continuous(
      labels = function(x) scales::comma(abs(x)),
      expand = expansion(mult = c(0.10, 0.15))
    ) +
    scale_fill_manual(
      values = cols,
      breaks = c(rev(lvls_right), lvls_left),
      drop = FALSE
    ) +
    labs(
      title    = title,
      subtitle = paste0(subtitle, "\n\u2190 ", arrow_left,
                        "          ", arrow_right, " \u2192"),
      x = x_label, y = NULL, fill = NULL
    ) +
    theme_minimal(base_size = base_size) +
    theme(
      legend.position  = "bottom",
      panel.grid.major.y = element_blank()
    )
}

albo_classify <- function(cel_5plus, cel_4plus, cel_2plus) {
  dplyr::case_when(
    cel_5plus >= 0.90 & cel_4plus >= 0.90 & cel_2plus >= 0.90 ~ "A",
    cel_5plus >  0.67 & cel_4plus >  0.80 & cel_2plus >  0.90 ~ "B",
    cel_5plus >  0.50 & cel_4plus >  0.67 & cel_2plus >  0.67 ~ "C",
    cel_5plus >  0.33 & cel_4plus >  0.50 & cel_2plus >  0.67 ~ "D",
    cel_2plus >  0.75                                          ~ "E",
    cel_2plus >  0.50                                          ~ "F",
    cel_2plus >  0.33                                          ~ "G",
    cel_2plus >  0.10                                          ~ "H",
    TRUE                                                        ~ "I"
  )
}

cel_geo_muni <- cel_geo |>
  # Use aggregate (non-stratified) rows only
  filter(!is.na(municipality), is.na(urban_rural)) |>
  mutate(

by_albo <- function(df, speak_col, parent_col, speaker_col) {
  df |>
    left_join(cel_geo_muni, by = "muni_code") |>
    filter(!is.na(albo_group)) |>
    group_by(albo_group) |>
    summarise(
      n_municipalities    = n(),
      n_total_speakers    = sum(.data[[speaker_col]], na.rm = TRUE),
      n_total_parents     = sum(.data[[parent_col]],  na.rm = TRUE),
      n_total_children    = sum(n_children_with_parent, na.rm = TRUE),
      n_speaking          = sum(.data[[speak_col]],   na.rm = TRUE),
      transmission_median = median(transmission_rate, na.rm = TRUE),
      transmission_mean   = mean(transmission_rate,   na.rm = TRUE),
      transmission_q25    = quantile(transmission_rate, 0.25, na.rm = TRUE),
      transmission_q75    = quantile(transmission_rate, 0.75, na.rm = TRUE),
      .groups = "drop"
    ) |>
    arrange(factor(albo_group, levels = c("A","B","C","D","E","F","G","H","I")))
}

The Albó-Romero classification captures a sharp gradient for Aymara and Quechua, but the pattern for Guaraní and other lowland languages is structurally different:

  • Aymara and Quechua follow the expected gradient — transmission collapses as CEL concentration falls. Group B: Aymara 40%, Quechua 54%. Groups F–I: Aymara near zero, Quechua 7–17%.
  • Guaraní shows near-zero transmission in Groups B–F (which are Andean municipalities where almost no Guaraní speakers live) and rises only in Groups G–H — the Chaco lowland municipalities where Guaraní communities are actually located. The Albó grouping is calibrated to Andean populations and does not capture Guaraní community strength.
  • Other indigenous languages show a similarly non-monotonic pattern, peaking in Groups D–E, which happen to include isolated lowland municipalities with strong small-language communities (e.g. Tsimane´, Chiquitano, Moxeño territory). The median near zero in several groups reflects that few municipalities in those groups have meaningful speaker populations for any single “other” language.
  • The consistently higher national rates for Guaraní (56%) and other languages (68%) relative to Aymara and Quechua are partly a selection effect: smaller languages survive in more isolated communities, and it is precisely those communities that remain in the dataset.
