macrolanguages_enhanced_sil.csv (459 rows × 12 columns):
  - macro_code: ISO 639-3 macrolanguage code
  - macro_name_sil: SIL reference name for macrolanguage
  - member_code: ISO 639-3 member language code
  - member_status: Active/Deprecated/Retired
  - in_linguameta: TRUE/FALSE flag for metadata presence
  - member_name_linguameta: Name from linguameta (if available)
  - member_name_sil: SIL reference name
  - member_endonym: Self-designation (if available)
  - member_n_speakers: Estimated speaker count
  - member_endangerment: Endangerment status
  - member_glottocode: Glottocode identifier
  - names_differ: TRUE if linguameta and SIL names differ

TABLE SCHEMA:
For each row (member language):
  - Row identifier: macro_code + member_code
  - Macro-level: name from SIL
  - Member-level: linguameta metadata + SIL name + name consistency check
