1. butterfly_helpers.R (400 lines, 5.2 KB)
   ─────────────────────────────────────────
   Core implementation with 6 exported functions:
   
   • make_butterfly()
     One-call wrapper for complete butterfly chart. Handles all steps:
     data preparation, bar position extraction, label placement, plotting.
     
   • prepare_butterfly_data()
     Aggregate counts, set up left/right divergence, create y-axis labels
     with totals. Returns prepared data + metadata.
     
   • extract_bar_positions()
     Use ggplot_build() to extract actual bar positions and segment
     midpoints. Avoids fragile cumsum() logic.
     
   • prepare_butterfly_labels()
     Filter segment labels into "inside" (wide bars) and "above" (small bars
     needing repel). Compute left/right totals.
     
   • make_butterfly_plot()
     Construct complete ggplot object with all aesthetics, annotations,
     themes, and legends. Fully customizable via parameters.
     
   • label_compact()
     Format numbers: 1234567 → "1.2M", 45000 → "45K", 123 → "123"

   Dependencies: tidyverse, ggplot2, ggrepel, scales
   
   Status: ✓ Tested with actual CEL data
   Quality: Production-ready, fully documented

MAINTENANCE NOTES
═════════════════════════════════════════════════════════════════════════════

If modifications needed in future:

1. Edit butterfly_helpers.R
   → Changes automatically apply to all downstream uses
   → No need to update multiple .qmd files

2. For new chart types:
   → Copy make_butterfly() call
   → Adjust parameters (left_cats, right_cats, colors, etc.)
   → No new code needed

3. For theme changes:
   → Modify make_butterfly_plot() theme section
   → All charts automatically updated

4. For new functionality:
   → Add to appropriate prepare_* function
   → Or create new function following same pattern
   → Update documentation
