## Project Standards

### File Organization
- **Rule**: Store all data files in `data/` subfolder
- **Standard**: Use `here::here()` for all file paths
- **Why**: Robust, survives file moves, cross-platform compatible

**See**: `.CODING_PREFERENCES.md` for detailed guidance

### Documentation
- Store complex analysis in `documentation/` as markdown files
- Reference from code with comments: `# See documentation/ANALYSIS_NAME.md`
- Examples: MACROLANGUAGES.md, WIKI_LANGUAGE_DIVERSITY_MACROLANGUAGE_ANALYSIS.md

### Code Style
- Use base R pipe `|>` (not `%>%`)
- Load `library(here)` in scripts that reference data
- Comment data file sources and usage

## Key Statistics & Findings

### Macrolanguages Merge
- **63 macrolanguages** with **459 member languages**
- **93.9%** coverage from linguameta metadata

- **102 total** languages with 0 Wikipedia articles
